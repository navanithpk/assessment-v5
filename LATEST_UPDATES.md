# Latest Updates - January 25, 2026

## Question Library Enhancements ✅

### 1. Pagination Added
- **20 questions per page** maximum
- Clean pagination controls with First/Prev/Next/Last buttons
- Page numbers displayed (e.g., "Page 2 of 15")
- Maintains filters and sorting when navigating pages

### 2. Sortable Column Headers
All columns now have sortable headers with toggle functionality:
- **ID** - Sort by question ID
- **Marks** - Sort by marks value
- **Type** - Sort by question type (MCQ, Theory, etc.)
- **Grade** - Sort by grade name
- **Subject** - Sort by subject name
- **Topic** - Sort by topic name
- **Year** - Sort by exam year

**How it works:**
- Click any column header to sort ascending ↑
- Click again to sort descending ↓
- Visual indicators show current sort direction
- Hover effect on sortable columns
- Default: Sort by ID descending (newest first)

### 3. Year Column Added
- New "Year" column displays exam year for each question
- Shows "—" for questions without year
- Sortable like other columns

## UI/UX Improvements

### Visual Indicators
- **Sortable columns**: Hover effect + ⇅ icon
- **Active sort**: ↑ or ↓ arrow on sorted column
- **Pagination**: Current page highlighted in blue
- **Disabled buttons**: Grayed out when at first/last page

## User Guide

### How to Use Pagination
1. Navigate to **Question Library**
2. Apply your filters (Grade, Subject, etc.)
3. Use pagination controls at bottom
4. Current page shows in blue

### How to Sort Questions
1. Click any column header to sort by that field
2. First click: Sort ascending (A→Z, 1→9)
3. Second click: Sort descending (Z→A, 9→1)
